09 · Preguntas frecuentes

Resolvemos sus dudas
antes de que las tenga

De cuantos modulos dispone Mindden Suite?

7 modulos integrados: Comercial (CRM), Financiero (facturacion, cobros, contratos), Proyectos y Operaciones, Recursos Humanos, Analitica e IA, Dashboard Ejecutivo y Automatizaciones. Cada modulo es independiente pero comparte datos en tiempo real.

Cuanto tiempo tarda la implementacion?

El despliegue tecnico se ejecuta en minutos con un push a GitHub Actions. La configuracion inicial (clientes, productos, roles) toma 1-2 dias. La migracion de datos existentes depende de la complejidad, pero el equipo operativo puede empezar a usar la plataforma desde el dia 1.

Mis facturas emitidas pueden editarse?

No. Mindden Suite garantiza la inmutabilidad financiera: las facturas emitidas no pueden modificarse. Cualquier correccion se realiza via notas de credito. Los datos fiscales (razon social, NIF/CIF) se congelan al momento de emision, preservando la integridad del registro contable.

Como detecta los impagos?

El semaforo de riesgo se actualiza automaticamente todos los dias a las 07:00. Si un cliente tiene 3 o mas facturas vencidas o 10.000 EUR o mas en impagos, se marca como ROJO y se bloquea automaticamente la facturacion para evitar ampliar la deuda.

Que tecnologia utiliza?

Python + Django 5 en backend, React 19 + TypeScript en frontend, PostgreSQL 17 como base de datos, Redis 7 para cache y colas, Celery para tareas asincronas, MinIO para almacenamiento de documentos, desplegadocon Docker y CI/CD.

Puedo escalar con el crecimiento de mi empresa?

Si. La arquitectura modular permite escalar cada componente de forma independiente. PostgreSQL soporta escalado vertical y read replicas. Los workers de Celery se escalan horizontalmente. Docker permite anyadir contenedores segun la demanda.
